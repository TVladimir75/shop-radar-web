Как открыть сайт
=================

1) Дважды щёлкните файл:
   Открыть_в_браузере.command

2) Если macOS пишет, что файл из интернета и не даёт открыть:
   ПКМ по файлу → Открыть → Открыть.

3) Откроется чёрное окно и (обычно) браузер.
   Точный адрес смотрите В ОКНЕ (если 8765 занят — будет 8766 и т.д.).
   Часто это: http://127.0.0.1:8765

   На Рабочем столе можно использовать ярлык:
   ПОСТАВЩИКИ_ОТКРЫТЬ.command

4) Окно Terminal НЕ закрывайте, пока смотрите сайт.
   Закрыли окно — сайт выключился.

Файл .command пишет «нет прав доступа»
--------------------------------------
В Терминале (замените путь на свой):

  chmod +x /Users/ВЫ/shop-radar-web/Открыть_в_браузере.command
  xattr -d com.apple.quarantine /Users/ВЫ/shop-radar-web/Открыть_в_браузере.command

(вторая команда нужна, если macOS пометил файл как скачанный из сети)

Не получается?
---------------
Откройте Терминал, выполните по очереди:

  cd "(перетащите сюда папку shop-radar-web)"
  ./.venv/bin/python start.py

Скопируйте текст ошибки из окна.

Если порт занят (Address already in use):
  закройте другие копии программы или перезагрузите Mac.

Доступ с телефона в той же Wi‑Fi сети
--------------------------------------
Перед запуском в Терминале:

  export SHOP_RADAR_HOST=0.0.0.0
  ./.venv/bin/python start.py

В чёрном окне появится второй адрес вида http://192.168.x.x:8765 — откройте его в браузере телефона.
